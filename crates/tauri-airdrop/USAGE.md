# Airdrop Tauri 应用

基于 Tauri 的跨平台局域网文件快传应用。

## 功能特性

- 🚀 **自动设备发现** - 基于 mDNS 的局域网设备自动发现
- 📤 **文件传输** - 点对点文件快速传输
- 💻 **跨平台** - 支持 macOS, Windows, Linux
- 🎨 **现代化 UI** - 基于 React + Tailwind CSS
- 🔔 **系统通知** - 文件接收时自动通知

## 开发模式

```bash
# 进入项目目录
cd crates/tauri-airdrop

# 安装依赖（如果还没安装）
npm install

# 启动开发服务器
npm run tauri dev
```

## 构建生产版本

```bash
npm run tauri build
```

## 使用说明

1. **启动应用** - 应用会自动初始化并开始扫描局域网设备
2. **选择设备** - 从左侧列表选择要发送文件的目标设备
3. **发送文件** - 点击"选择文件发送"按钮，选择文件
4. **接收文件** - 文件会自动保存到下载目录，并显示系统通知
